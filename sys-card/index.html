01 — Design Principles

Modularity

Cards assemble from discrete blocks — header, image, data grid, list, actions, body. Blocks are interchangeable and order-independent. Add what you need, omit what you don't. The system composes; it never prescribes.

Proportion

All spacing derives from a single 8px unit. Padding is 2u. Gaps are 1px. Type steps at 10, 12, 14, 15. No magic numbers. Every measurement traces to the base unit, producing consistent rhythm across all configurations.

Grid

The root card is a CSS Grid with row-gap: 1px over a foreground-colored background. Children fill with the background color. The gap reveals the line. Nested modules repeat this pattern. One mechanism, zero border doubling.

Contrast

Two values: foreground and background. Lines, text, and loading states use foreground. Surfaces use background. Grayscale images enforce tonal unity. Swap two CSS custom properties and every element follows.
